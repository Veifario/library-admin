## Usage

Install dependencies for client side:

```sh
$ cd ./client
```

```sh
$ npm install
```

Install dependencies for server side:

```sh
$ cd ./server
```

```sh
$ npm install
```

Run client:

```sh
$ npm run dev
```

Run server:

```sh
$ npm start
```
